## IF statements (conditions)
The simplest control scturcture is the if statement, which executes code depending a condition being true or false. Consider the following sample:
~~~
examMark = 68

if examMark >= 50:
    print("Pass")
else:
    print("Fail")
~~~

### Multiple conditions using Elif (else if)
~~~
if balance < 0:
    print("Trouble!")
elif balance == 0:
    print("Start saving")
else:
    print("There is something")
~~~
